# 抖音 AI 带货工作台

零 API 费用优先的抖音带货视频生产工作台 MVP。

## 已实现

- 商品资料与事实证据分级
- 商品图、参考视频与链接管理
- 浏览器本地视频关键帧提取
- 三种差异化带货剧本生成
- 逐镜分镜与豆包/千问视频提示词
- Markdown 生产任务包导出
- 浏览器本地自动保存

## 费用策略

工作台不主动调用付费 API。素材整理、关键帧提取、脚本模板、分镜和导出均在浏览器本地完成。画面语义分析、参考图和视频生成可继续使用已有 ChatGPT、豆包或千问会员额度手动完成。

## 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:3000 。

## 安全

不要向仓库提交账号密码、Cookie、API Key 或其他密钥。商品价格、功效和适用范围必须以商品页及实测证据为准。
